# Password Manager (Python)

This is a simple command-line based password manager created using Python.  
The project allows a user to store and manage login details for different websites using a master password.

---

## About the Project

The main idea of this project is to avoid storing passwords in plain text.  
A master password is used to control access, and all data is stored in a local file.  
The project is kept simple and focuses only on core Python concepts taught in the semester.

---

## Tools and Technologies
- Python  
- File handling  
- Built-in Python libraries  

---

## Features
- Master password based access  
- Add website login details  
- Store data in a file  
- View saved records  
- Simple menu-driven program
